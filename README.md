# Quizzing App

## Installation & Setup Guide
1. Clone the repository
```
git clone https://github.com/CyberKyoujin/quizzing-app.git
```
2. Run tests
```
./gradlew connectedAndroidTest   
```
3. If tests were successful, run the application
